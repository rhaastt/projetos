Propriedades Herdadas

Existe três técnicas para usar o CSS:
A ordem é:

CSS inline

São usados para alterações pequenas e exclusivas de um unico elemento. De preferencia em imagens especiais, cores de elementos especificos, tamanho desses elementos.É bastante usado em itens com tempo de vida.

CSS interno/local

É uma forma não recomendada para projetos mais complexos e com mais de uma pagina. Ele é utilizado dentro do próprio projeto html com a tag STYLE no head, é bom para paginas unicas ou para teste.

CSS externo

São folhas criados em um link separados. São usadas para criar um estilo padrão em mais de uma pagina deixando o projeto mais organizado e profissional, possibilitando alterações completas e de um jeito mais facil.

Se caso ocorrer de uma CLASS e um ID estiverem no mesmo elemento vai ocorrer do primeiro sobreescrever o outro e caso isso aconteça pode ser usado a técninca INLINE, ela é a quem tem o mais peso.
