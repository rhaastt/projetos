Seletores Personalizados em CSS

Como Utilizar

Para dar mais poder as CSS, criando estilos personalizados é preciso utilizar DIVS os seletores ID (#) e CLASS (.) de maneira eficiente.
Podemos indentificar um determinado elemento único com ID, ou agrupar elementos multiplos que tenham características unicas semelhantes com um CLASS.

Exemplo:

Quando usar ID

O ID vai indentificar um elemento único dentro da pagina atual. E isso permiti criar um estilo personalizado para um elemento isolado.

Quando usar CLASS

A CLASS vai indentificar uma classe à qual um ou mais elementos pertençam, compartilhando caracteristicas em comum a todos os que façam parte desse grupo.
