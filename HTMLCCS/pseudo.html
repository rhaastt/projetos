Pseudo-classes e Pseudo-elementos

Pseudo-classes

É uma palavra-chave que adiciona um estado especial a um elemento, ela se posiciona logo após do sinal de dois pontos
exemplo: .classe:hover
São elas:

  • hover
  • visited
  • active
  • checked
  • empty
  • focus

Pseudo-elementos

São palavra-chaves adicionada ás declarações de um seletor através de dois sinais de dois pontos e permite que formate um pedaço especifico do elemento referenciado.
Os principais são:

  • ::before

    • É utilizado para acrescentar algo antes

  • ::after

    • É utilizado para acrescentar algo depois

  • ::first-letter

    • É utilizado na primeira letra do elemento

  • ::first-inline

    • É utilizado na primeira linha do elemento
